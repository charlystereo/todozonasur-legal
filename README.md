# Política de Privacidad - TodoZonaSur

Última actualización: 27 de abril de 2026

## Quiénes somos
TodoZonaSur es una página informativa sobre noticias locales de la zona sur 
del conurbano bonaerense (Argentina). Operamos a través de Facebook.

## Información que recolectamos
TodoZonaSur NO recolecta, almacena ni procesa datos personales de los 
usuarios que interactúan con nuestra página de Facebook. Toda interacción 
queda gestionada por Meta/Facebook según sus propias políticas.

Las únicas operaciones que realizamos son:
- Publicar contenido informativo en nuestra página de Facebook
- Recopilar noticias públicas de medios locales para curar contenido

## Información que NO recolectamos
- Datos personales identificables
- Direcciones de email
- Números de teléfono
- Direcciones físicas
- Información financiera

## Uso de Facebook Graph API
Utilizamos la API oficial de Facebook únicamente para publicar contenido 
en nuestra propia página. No accedemos a datos de usuarios, no leemos 
información personal de seguidores, no enviamos mensajes privados.
